### What I learned

Learn the fundamentals of CSS grid and how to use grid-col and grid-row to adjust content position in media queries.


```css
/* patrick */
 .black-blue-card {
  grid-column: 1;
  grid-row: 4;
 }

 /* kira */
 .white-card-kira {
  grid-column: 1;
  grid-row: 5;
 }
```
